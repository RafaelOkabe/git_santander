# Cuso Digital: Git

## Comandos
-> git init\
-> git config --global user.name "name"\
-> git config --global user.email email\
-> git status\
-> git add .\README.md\
-> git diff\
-> git diff --staged\
-> git commit -m "add new lines"\
-> git log\
-> git restore .\README.md\
-> git restore --staged \README.md\
-> git remote\
-> git push origin master
